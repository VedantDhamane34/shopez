User Flow
The user flow describes the sequence of actions a user performs while interacting with the ShopEZ platform. It illustrates how users navigate through the system from account creation to completing a purchase.
1. User Registration / Login
The user first registers on the platform by providing basic details such as name, email, and password. Existing users can log in using their credentials.
2. Browse Products
After logging in, the user can browse the product catalog. Products are organized into categories to help users find items easily.
3. View Product Details
Users can click on a product to view detailed information including description, price, rating, available stock, and reviews.
4. Add Product to Cart
If the user decides to purchase a product, they can add it to their shopping cart.
5. Review Cart
Users can review the items added to their cart, modify quantities, or remove products before proceeding to checkout.
6. Checkout
The user proceeds to checkout where they confirm their shipping details and select a payment method.
7. Payment Processing
The payment is processed through the system. Once the payment is successful, the order is confirmed.
8. Order Confirmation
The system generates an order record and sends confirmation to the user. The user can also view their order history.
