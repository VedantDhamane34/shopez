# Features

ShopEZ is a full-stack e-commerce platform built using the MERN stack (MongoDB, Express.js, React.js, Node.js). The system provides a seamless online shopping experience for customers while offering efficient management tools for administrators.

## User Features

User Registration and Login  
Users can create an account and securely log in to the platform using their credentials.

Product Browsing  
Users can browse products across multiple categories and explore the available catalog.

Product Details  
Users can view detailed product information including description, price, images, ratings, and reviews.

Shopping Cart  
Users can add products to their shopping cart, update quantities, or remove items before checkout.

Secure Checkout  
Users can proceed to checkout and place orders through a secure transaction process.

Order History  
Users can view previously placed orders and track their purchase history.

Product Reviews and Ratings  
Users can leave reviews and ratings for products they have purchased.

## Admin Features

Product Management  
Admins can add new products, update existing product information, and remove products from the catalog.

Category Management  
Admins can create and manage product categories to organize the product catalog efficiently.

Order Management  
Admins can monitor and manage customer orders including order status and fulfillment.

User Management  
Admins can view registered users and manage platform activity.

## System Features

RESTful API Architecture  
The platform uses REST APIs to enable communication between the frontend and backend.

Secure Authentication  
Authentication is implemented using JSON Web Tokens (JWT) to protect user sessions.

Password Encryption  
User passwords are securely stored using bcrypt encryption.

Database Management  
MongoDB is used as the primary database for storing user, product, order, and review data.

Responsive User Interface  
The frontend is built with React to provide a dynamic and responsive user experience.
